Resumo

O Mapa do Cidadão é uma ferramenta de e-Gov e Smart Cities para o cadastro de ocorrências municipais, seu intuito é melhorar a cidadania e promover transparência para os dados de problemas municipais, utiliza o facebook como autenticação para diminuir o número de contas falsas aumentando a veracidade dos dados.

Tem um sistema interno de Votos e Vetos para dar relevância ao dado, além de um sistema contra SPAMs que esconde ocorrências possivelmente falsas.

Usa também o facebook para promover o compartilhamento das ocorrências para tornar o Mapa do Cidadão uma microrrede social.
